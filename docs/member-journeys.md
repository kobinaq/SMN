# Member journeys

## First-time member

1. Register at `/register`
2. Sign in at `/login`
3. Land on `/app` continuity dashboard
4. Complete essential profile (non-blocking checklist)
5. Open Learning → start first unlocked course
6. Discover Mentors and Opportunities from secondary cards

## Resume learning

1. Open `/app`
2. Primary CTA links to the last in-progress or next incomplete lesson
3. Mark lesson complete — UI confirms save or offers retry on failure
4. Continue with previous/next lesson controls

## Mentorship

1. Browse `/app/mentors`
2. Submit a request (duplicate prevention enforced server-side)
3. Track status from mentors area / dashboard secondary card

## Opportunities

1. Browse `/app/opportunities` with filters
2. Open a listing; expired listings are labelled and not active apply targets
3. Track personal application status (manual tracking — no auto-apply)

## Portfolio & certificates

1. Create a draft case study at `/app/portfolio`
2. Preview public presentation before publish
3. View issued certificates at `/app/certificates` and share verification links
